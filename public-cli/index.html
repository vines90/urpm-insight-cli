URPM Insight CLI

TCL URPM 用户洞察平台 · 命令行工具 + Agent Skills（Cursor / Trae / Claude / Codex / Continue）

一键安装（推荐）

Skills 默认只写入你指定的那一个 Agent，避免往 Cursor / Trae 等全部拷贝。请把下面命令里的 trae-cn 换成你实际在用的 Agent：

URPM_AGENT=trae-cn curl -fsSL https://urpm.tclac.com/cli/install.sh | bash

可选值：cursor · claude · trae · trae-cn · codex · continue · agents · all（全部目录）

若本机只装了一个 Agent，也可不写 URPM_AGENT，脚本会自动装到那一个；装了多个且猜不到当前终端归属时，会提示你显式指定。

install.sh urpm-insight-cli.tar.gz 📖 完整指南（飞书）

首次使用

urpm auth login
urpm --help
urpm dashboard overview

其它用法

# 装到全部 Agent 目录（旧行为）
URPM_AGENT=all curl -fsSL https://urpm.tclac.com/cli/install.sh | bash

# 只装 CLI，不复制 skills
SKIP_SKILLS=1 curl -fsSL https://urpm.tclac.com/cli/install.sh | bash

# 自定义 skills 目录（覆盖 URPM_AGENT）
SKILLS_DESTS=~/my/skills curl -fsSL https://urpm.tclac.com/cli/install.sh | bash

# 内网证书：加 -k
URPM_AGENT=trae-cn curl -kfsSL https://urpm.tclac.com/cli/install.sh | bash
